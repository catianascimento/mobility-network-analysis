# Mobility Network Analysis

Projeto para análise de mobilidade utilizando redes e notebooks Jupyter.

## Estrutura esperada

O projeto espera a seguinte estrutura de pastas:

```text
project/
│
├── mobility_data/
├── maps/
├── analise_mobilidade.ipynb
├── environment.yml
└── README.md
```

As pastas abaixo não estão incluídas no repositório e devem ser adicionadas manualmente:

- `mobility_data/`
- `maps/`

## Instalação do ambiente

Crie o ambiente Conda utilizando:

```bash
conda env create -f environment.yml
```

Ative o ambiente:

```bash
conda activate NOME_DO_AMBIENTE
```

## Execução

Abra o notebook:

```bash
jupyter notebook
```

E execute:

```text
analise_mobilidade.ipynb
```
